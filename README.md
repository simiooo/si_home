# Si Home
A Chrome New Tab extension.
## Requirements
* nodejs >= 19
* pnpm >= 10
## build
1. Insall Dependency .
```
pnpm i
```
2. Build

* For Firefox
``` 
pnpm build:firefox
```
* For Chrome
```
pnpm build
```

3. Get output of build
```
<current_dir>/dist
```
or
```
<current_dir>/dist-firefox
```
